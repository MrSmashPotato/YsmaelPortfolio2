Niis Restaurant Menu is an Android application built using .NET MAUI and C#. This app allows customers to browse the menu and place orders seamlessly. One of its standout features is real-time updates to the admin dashboard, which provides instant visibility into customer orders as they are placed. The user interface is designed to be simple and intuitive, requiring no login from customers, making it easy for anyone to use.

Technologies Used: .NET MAUI, C#, Android
